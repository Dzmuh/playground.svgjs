Пицца на SVG.js
===============

По мотивам [JavaScript Pizza by Cassidy Williams](https://github.com/cassidoo/javascript-pizza)

Что надо:

```
npm install bower -g
npm install typescript -g
npm install tsd -g
```

А ещё Visual Studio 2015, желательно.

Собираем:

```
bower install
tsd install
tsc pizza.ts
```
